# mini-is

## About

Small and Beautiful JS Type Judgment Method Library

## Installation

```bash
# pnpm
pnpm i mini-is

# yarn
yarn add mini-is

#npm
npm i mini-is
```

## Usage

```js
// ESM xxx.js
import { isString } from "mini-is";
console.log(isString("hello"));

// CJS xxx.js
const { isString } = require("mini-is");
console.log(isString("hello"));
```
